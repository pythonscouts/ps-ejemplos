# Ejemplos de Python Scouts

Este repositorio reúne ejemplos de código en Python usados en los contenidos de Python Scouts. Incluye material introductorio y práctico sobre fundamentos del lenguaje, palabras clave, estructuras de control, funciones, manejo de errores, asincronía y pequeñas aplicaciones con librerías como FastAPI, Tkinter, Kivy, PyQt y PySide6.

El contenido está organizado en tres áreas principales:

- [`Series`](./Series): colecciones de ejemplos cortos por tema o palabra clave de Python, con archivos `repl_*.py` y ejercicios prácticos.
- [`Guías`](./Guías): ejemplos puntuales para resolver tareas concretas, como ejecutar scripts, crear archivos ZIP o revisar elementos repetidos en listas.
- [`Tutoriales`](./Tutoriales): proyectos y demos un poco más amplios para interfaces gráficas y desarrollo de APIs.
